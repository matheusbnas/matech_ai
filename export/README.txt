# Site Matech.AI

Este pacote contém todos os arquivos necessários para implantar o site da Matech.AI em um servidor.

## Conteúdo do Pacote:

- `/frontend/` - Arquivos estáticos do frontend (prontos para produção)
- `/backend/` - Código-fonte do backend em FastAPI
- `GUIA_IMPLANTACAO.md` - Guia detalhado de implantação em diferentes servidores

## Requisitos mínimos:

- Python 3.8+
- MongoDB
- Servidor web (Nginx, Apache, etc.)

## Instruções Rápidas:

1. Leia o arquivo `GUIA_IMPLANTACAO.md` para instruções detalhadas
2. Escolha o método de implantação mais adequado para sua infraestrutura
3. Configure o backend e o frontend conforme as instruções
